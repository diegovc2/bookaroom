Setup Help!

Setup steps after installing the plugin.

  1. Edit main settings

    Go to the Meeting Room Settings page.
    1. Base increment in minutes
      How many minutes in a schedulable block. Entering 30 means you can schedule rooms in hald-hour blocks.

    2. Setup and Cleanup increments
      This is the number of increments (from above) you would like to add as a buffer before and/or after each meeting for setup and cleanup. This adds one increment before and/or after each reservation that is unavailable to reserve, allowing staff to have time to setup or cleanup the room.

    3. URL for reservation page
      This is a page (at least one) that you have created and would like to use to display the plugin.

      To embed the plugin on a page, you would add the shortcode [meetingRooms].
